### 📌 Project concept

1. **영화 소개**
    1. 리스트 나열, 즐겨 찾기, 회원 가입, 로그인(open source API 사용)
    2. 페이지 분류
        1. 최신 영화 리스트
        2. 영화 검색
        3. 커뮤니티
        4. 사용자 정보
